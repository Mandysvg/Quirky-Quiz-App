# Quirky-Quiz-App
A simple Python quiz program to test your general knowledge with multiple choice questions.

How to play
1. Read each question carefully
2. Type A, B, or C to answer
3. Get instant feedback after you answer
4. See your final score at the end

Example 
Welcome to the quirky quiz app :)

Question # 1
How many official languages does South Africa have?
A. 11 official languages
B. 12 official languages
C. 10 official languages
Please answer the question('A','B', 'C')> A
Correct! 

Quiz Completed
Your score is: 1/1
